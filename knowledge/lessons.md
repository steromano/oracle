# Lessons

Accumulated, numbered, **process-tagged** lessons mined by the `retrospect`
skill (§5.12, Channel 1). A lesson is valid only if it is (a) actionable,
(b) general beyond one question, and (c) tagged to a **process defect
observable at forecast time** — never derived from a surprising outcome
("resulting"). Outcomes feed only the statistical recalibration fit, never
lessons.

Format: `L-NNN [tag]: <actionable, general lesson>` where `tag` is one of the
process-checklist categories (spec, research, base-rates, modelling, ensemble,
red-team, arithmetic, evidence-dates).

Read this file before every forecast (CLAUDE.md hard rule 3) and cite lesson
numbers where they influence the forecast.
